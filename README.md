# Portfolio Website

A modern and responsive portfolio website showcasing skills, projects, and contact information.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Smooth Navigation**: Smooth scrolling between sections
- **Modern UI**: Clean and professional design with animations
- **Interactive Elements**: Hover effects and scroll animations
- **Contact Form**: Simple contact form for visitor inquiries

## Sections

- **Home**: Welcome section with hero banner
- **About**: Personal introduction and background
- **Skills**: Showcase of technical skills and expertise
- **Projects**: Portfolio of completed projects
- **Contact**: Contact form for inquiries

## Usage

Simply open `index.html` in a web browser to view the portfolio website.

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)

## Customization

To customize the portfolio:

1. Edit `index.html` to update content (name, bio, projects, etc.)
2. Modify `style.css` to change colors, fonts, and styling
3. Update `script.js` to add or modify interactive features
